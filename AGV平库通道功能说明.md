# AGV平库通道管理功能说明

## 功能概述

AGV平库通道管理是WMS立体库管理系统中的核心功能模块，专门用于管理AGV自动化平库的通道配置、运营控制和任务分配管理。

## 菜单位置

- **主菜单**: 仓库管理
- **子菜单**: AGV平库通道

## 功能特性

### 1. 数据展示
- **通道编码**: 通道的唯一标识符
- **通道名称**: 便于识别和管理的名称
- **所属库区**: 关联的库区信息
- **库位结构**: 支持单深/双深两种存储结构
- **状态**: 启用/禁用状态控制
- **备注**: 附加说明和描述信息
- **创建时间**: 记录创建的时间戳

### 2. 查询功能
- **通道编码查询**: 支持模糊匹配搜索
- **通道名称查询**: 支持模糊匹配搜索
- **所属库区筛选**: 支持按库区精确筛选
- **状态筛选**: 支持按状态过滤数据
- **组合查询**: 支持多条件同时查询
- **重置功能**: 一键清空查询条件
- **实时过滤**: 查询结果即时更新

### 3. 操作功能
- **新增通道**: 创建新的AGV平库通道
- **编辑通道**: 修改现有通道信息（通道编码不可修改）
- **删除通道**: 移除通道记录（需确认）
- **启用通道**: 恢复正常任务分配
- **禁用通道**: 停止任务分配（特殊确认提示）

### 4. 特殊业务逻辑
- **编辑限制**: 编辑模式下通道编码字段自动禁用
- **禁用确认**: 禁用时显示特殊警告："禁用后堆垛机将不再执行该巷道的出入库任务，是否确认？"
- **启用恢复**: 启用后恢复正常任务分配
- **唯一性验证**: 通道编码必须唯一
- **关联验证**: 必须选择所属库区

## 使用说明

### 1. 查询操作
1. 在筛选区输入通道编码或名称
2. 选择所属库区和状态筛选条件
3. 点击"查询"按钮执行搜索
4. 使用"重置"按钮清空查询条件

### 2. 新增操作
1. 点击"新增通道"按钮
2. 在弹窗中填写通道编码（必填、唯一）
3. 填写通道名称（必填）
4. 选择所属库区（必填）
5. 选择库位结构（必填）
6. 填写备注信息（可选）
7. 点击"保存"完成创建

### 3. 编辑操作
1. 点击数据行中的"编辑"按钮
2. 在弹窗中修改相关信息（通道编码不可修改）
3. 点击"保存"确认修改

### 4. 删除操作
1. 点击数据行中的"删除"按钮
2. 在确认对话框中确认删除
3. 数据将从列表中移除

### 5. 状态切换
#### 禁用操作
1. 点击数据行中的"禁用"按钮
2. 系统显示特殊确认对话框
3. 确认信息："禁用后堆垛机将不再执行该巷道的出入库任务，是否确认？"
4. 确认后通道状态变为禁用

#### 启用操作
1. 点击数据行中的"启用"按钮
2. 在确认对话框中确认操作
3. 提示："启用后将恢复正常任务分配"
4. 确认后通道状态变为启用

## 业务规则

### 1. 数据验证规则
- **通道编码**: 必填，唯一性验证
- **通道名称**: 必填
- **所属库区**: 必填，必须从现有库区中选择
- **库位结构**: 必填，只能选择单深或双深
- **状态**: 默认启用

### 2. 操作权限规则
- **编辑限制**: 通道编码在编辑模式下不可修改
- **删除限制**: 删除操作需要确认
- **状态切换**: 禁用操作需要特殊确认

### 3. 业务逻辑规则
- **任务分配**: 禁用状态的通道不参与任务分配
- **堆垛机控制**: 禁用后堆垛机停止执行该通道的出入库任务
- **恢复机制**: 启用后立即恢复正常任务分配

## 注意事项

1. **数据持久化**: 当前版本使用内存存储，页面刷新后数据会重置
2. **权限控制**: 实际使用时需要配置相应的用户权限
3. **API集成**: 生产环境需要对接后端API接口
4. **任务影响**: 禁用通道会直接影响堆垛机任务执行
5. **数据一致性**: 需要确保通道与库区数据的一致性
