# AGV平库库位管理功能说明

## 功能概述

AGV平库库位管理是WMS立体库管理系统中的核心功能模块，专门用于管理AGV自动化平库的库位状态、容器信息、站点配置和运营控制。

## 菜单位置

- **主菜单**: 仓库管理
- **子菜单**: AGV平库库位

## 功能特性

### 1. 数据展示
- **库位编码**: 库位的唯一标识符
- **所属库区**: 关联的库区信息
- **通道**: 关联的通道信息
- **当前状态**: 空库位/有货/已分配/空托
- **锁定状态**: 锁定/正常
- **启用状态**: 启用/禁用
- **当前容器号**: 如有货时显示容器编号
- **是否AGV站点**: 标识是否为AGV操作站点
- **站点类型**: 入库站点/出库站点/缓存站点/检测站点

### 2. 查询功能
- **库位编码查询**: 支持模糊匹配搜索
- **所属库区筛选**: 支持按库区精确筛选
- **通道筛选**: 支持按通道精确筛选
- **当前状态筛选**: 支持按库位状态筛选
- **锁定状态筛选**: 支持按锁定状态筛选
- **组合查询**: 支持多条件同时查询
- **重置功能**: 一键清空查询条件
- **实时过滤**: 查询结果即时更新

### 3. 操作功能
- **新增库位**: 创建新的AGV平库库位
- **编辑库位**: 修改现有库位信息
- **删除库位**: 移除库位记录（需确认）
- **启用/禁用**: 控制库位可用性
- **锁定/解锁**: 控制库位锁定状态

### 4. 业务逻辑验证
- **状态关联验证**: 有货状态必须填写容器号
- **站点类型验证**: AGV站点必须选择站点类型
- **字段联动**: 非AGV站点不能选择站点类型
- **唯一性验证**: 库位编码必须唯一
- **必填字段验证**: 确保关键信息完整

## 状态管理

### 1. 当前状态类型
- **空库位**: 库位为空，可以存放货物
- **有货**: 库位已存放货物，需要容器号
- **已分配**: 库位已被分配但未存货
- **空托**: 库位存放空托盘

### 2. 锁定状态控制
- **正常**: 库位可正常使用
- **锁定**: 库位被锁定，暂停使用

### 3. 启用状态管理
- **启用**: 库位正常运营
- **禁用**: 库位停止使用

## AGV站点管理

### 1. 站点类型
- **入库站点**: 用于货物入库操作
- **出库站点**: 用于货物出库操作
- **缓存站点**: 用于临时存储缓存
- **检测站点**: 用于货物质量检测

### 2. 站点配置规则
- AGV站点必须配置站点类型
- 非AGV站点不能配置站点类型
- 站点类型影响AGV任务分配

## 使用说明

### 1. 查询操作
1. 在筛选区输入库位编码
2. 选择所属库区、通道、状态等筛选条件
3. 点击"查询"按钮执行搜索
4. 使用"重置"按钮清空查询条件

### 2. 新增操作
1. 点击"新增库位"按钮
2. 填写库位编码（必填、唯一）
3. 选择所属库区和通道（必填）
4. 设置当前状态，如选择"有货"需填写容器号
5. 选择是否为AGV站点，如是则必须选择站点类型
6. 设置启用状态和备注信息
7. 点击"保存"完成创建

### 3. 编辑操作
1. 点击数据行中的"编辑"按钮
2. 在弹窗中修改相关信息
3. 注意业务逻辑验证规则
4. 点击"保存"确认修改

### 4. 删除操作
1. 点击数据行中的"删除"按钮
2. 在确认对话框中确认删除
3. 数据将从列表中移除

### 5. 状态控制
#### 启用/禁用操作
1. 点击数据行中的"启用/禁用"按钮
2. 确认操作后状态立即生效

#### 锁定/解锁操作
1. 点击数据行中的"锁定/解锁"按钮
2. 确认操作后锁定状态立即生效

## 业务规则

### 1. 数据验证规则
- **库位编码**: 必填，唯一性验证
- **所属库区**: 必填，必须从现有库区中选择
- **通道**: 必填，必须从现有通道中选择
- **当前状态**: 必填，默认为空库位
- **容器号**: 有货状态时必填
- **站点类型**: AGV站点时必填

### 2. 状态控制规则
- **锁定状态**: 锁定的库位不能进行货物操作
- **启用状态**: 禁用的库位不参与任务分配
- **AGV站点**: 影响AGV任务路径规划

### 3. 业务逻辑规则
- **容器关联**: 有货状态必须有容器号
- **站点配置**: AGV站点必须配置站点类型
- **状态一致性**: 确保库位状态与实际情况一致

## 注意事项

1. **数据持久化**: 当前版本使用内存存储，页面刷新后数据会重置
2. **权限控制**: 实际使用时需要配置相应的用户权限
3. **API集成**: 生产环境需要对接后端API接口
4. **状态同步**: 需要与AGV系统保持状态同步
5. **数据一致性**: 确保库位、库区、通道数据的一致性
