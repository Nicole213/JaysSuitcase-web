# AGV平库库区管理功能说明

## 功能概述

AGV平库库区管理是WMS立体库管理系统中新增的功能模块，专门用于管理AGV自动化平库的库区配置和运营管理。

## 菜单位置

- **主菜单**: 仓库管理
- **子菜单**: AGV平库库区

## 功能特性

### 1. 数据展示
- **巷道编码**: 库区的唯一标识符
- **巷道名称**: 便于识别和管理的名称
- **库位结构**: 支持单深/双深两种存储结构
- **绑定库台**: 关联的库台设备信息
- **绑定库排**: 关联的库排位置信息
- **状态**: 启用/禁用状态控制
- **备注**: 附加说明和描述信息
- **创建时间**: 记录创建的时间戳

### 2. 查询功能
- **巷道编码查询**: 支持模糊匹配搜索
- **巷道名称查询**: 支持模糊匹配搜索
- **状态筛选**: 支持按状态过滤数据
- **组合查询**: 支持多条件同时查询
- **重置功能**: 一键清空查询条件
- **实时过滤**: 查询结果即时更新

### 3. 操作功能
- **新增**: 创建新的AGV平库库区
- **编辑**: 修改现有库区信息
- **删除**: 移除库区记录（需确认）
- **启用/禁用**: 切换库区运营状态
- **状态管理**: 可视化状态标签显示

### 4. 数据验证
- **必填字段验证**: 巷道编码、名称、库位结构、状态
- **唯一性验证**: 巷道编码不能重复
- **格式验证**: 确保数据格式正确
- **实时反馈**: 验证错误即时提示

## 使用说明

### 1. 查询操作
1. 在筛选区输入巷道编码或名称
2. 选择状态筛选条件
3. 点击"查询"按钮执行搜索
4. 使用"重置"按钮清空查询条件

### 2. 新增操作
1. 点击"新增库区"按钮
2. 在弹窗中填写必要信息
3. 选择库位结构和绑定关系
4. 点击"保存"完成创建

### 3. 编辑操作
1. 点击数据行中的"编辑"按钮
2. 在弹窗中修改相关信息
3. 点击"保存"确认修改

### 4. 删除操作
1. 点击数据行中的"删除"按钮
2. 在确认对话框中确认删除
3. 数据将从列表中移除

### 5. 状态切换
1. 点击数据行中的"启用/禁用"按钮
2. 在确认对话框中确认操作
3. 状态标签将更新显示

## 注意事项
